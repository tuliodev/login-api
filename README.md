# Login API RESTFul

# Tools

- NodeJs
- Knex
- Postgresql
- Dotenv-safe
- Pg
- Express 
- Typescript
- JsonWebToken
- Bcrypt

# Features

- User registration system
- User login system
- User list system
- User filter system

# How to use 

$ git clone https://github.com/tuliodev/login-api.git

$ npm instal

Edit src/database/connection.ts

npm run knex:migrate

$ npm run dev

Access the application at http://localhost:3333
